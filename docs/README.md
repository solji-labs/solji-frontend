## 🏗️ 项目技术栈

- **前端框架**: Next.js 15.2.4 (React 19)
- **包管理器**: pnpm 9.15.0
- **区块链**: Solana Web3.js + Anchor
- **UI 组件**: Radix UI + TailwindCSS
- **钱包集成**: Solana Wallet Adapter
- **容器化**: Docker + Docker Compose
